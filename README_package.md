# NiCr MD docs and example templates package

This package contains:

- lightly cleaned public-facing example versions of the four input templates
- updated Markdown pages for the GitHub `docs/` folder

The template edits are intentionally minimal:
- private absolute paths were replaced by variables like `${potential_file}`
- original variable names and workflow logic were preserved as much as possible

## Included templates

- `templates/in.Step1_latticeConst_example.in`
- `templates/in.Step2_GridSearch_example.in`
- `templates/in.Step3_EqSigma3_112TB_example.in`
- `templates/in.Step4_ECO_example.in`
